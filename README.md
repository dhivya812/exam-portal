# 📝 Online Exam Portal

## 🔹 Overview
The Online Exam Portal is a web-based application that allows users to take a timed quiz and view their results instantly. This project demonstrates client-side logic, timer functionality, and dynamic score calculation.

---

## 🚀 Features
- 👤 User login (basic input)  
- ⏳ Timer-based exam system  
- ❓ Multiple choice questions  
- 📊 Automatic score calculation  
- 💾 Data stored using Local Storage  
- 🔄 Restart exam functionality  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript
